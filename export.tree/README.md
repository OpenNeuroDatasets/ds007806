## Examining excitation-inhibition modulation during social processing using functional Magnetic Resonance Spectroscopy

### Description
This dataset contains T1-weighted images, resting and functional MRS, and fMRI data collected as part of 
a study investigating the effect of dynamic face stimuli on GABA and Glx in social brain vs non-social brain. Data were acquired at Swinburne Neuroimaging, Australia 
using a 3T Siemens Prisma scanner.

### Dataset Description
- N = 40 participants (20 female, 20 male; age range 18–40, mean 30.1 ± 5.8 years)
- Inclusion criteria: self-reported fluency in written and spoken English, normal or corrected-to-normal vision, normal hearing, and no intellectual disability.
- Exclusion criteria: psychiatric, developmental, substance use, genetic, and neurological conditions (based on self-report). At the time of testing, participants were free from psychoactive medications (>1 month), nicotine (abstained >12 hrs), caffeine (abstained >12 hrs), alcohol (abstained >12 hrs), and recreational drugs (abstained >1 week). 

### Scanner & Acquisition
- Scanner: Siemens Prisma 3T
- T1w: MPRAGE, 1mm isotropic, TR=1900ms, TE=2.34ms
- MRS: HERMES, TR=2000ms, TE=80ms, 780 transients
- fMRI: EPI, 3mm isotropic, TR=2500ms, TE=30ms, 46 slices

### Notes
- MRS voxels: right “social brain” incl. TPJ, STG, and insular cortex (40x25x25mm) and control region predominantly within visual cortex (30x30x30mm)
- MRS tasks: 1st rest block (TRs 1-152), 1st functional block (TRs 153-304, 2nd rest block (TRs 305-546) 2nd functional block (TRs 547-698), 3rd rest block (TRs 699-780)
- All data have been defaced using pydeface prior to sharing.
